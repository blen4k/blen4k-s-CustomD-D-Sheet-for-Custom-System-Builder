# Blen4k`s Sheet Custom System Builder Foundry VTT - База для гибкого листа персонажа и системы в целом для сильно модифицированной D&D 5e.
Долгое время я искал удобный и гибкий лист персонажа для игры в **Dungeons & Dragons 5 редакции** на Foundry VTT, но ничего подходящего не нашёл.
Поэтому я решил создать свой собственный лист с помощью **Custom System Builder**.  

Этот репозиторий это мой шаблон системы, который я планирую использовать сильно модифицированную версию D&D 5e. 

### Почему именно Custom System Builder?
- Полноценные системы (например, официальная D&D5e) слишком жёсткие и плохо подходят под сильные домашние правки.
- Custom System Builder позволяет в любой момент добавлять новые элементы, поля, шкалы и механики без переписывания всей системы.
- Идеально, когда надо добовлять свои опциональные правила да и когда кампания постоянно эволюционирует.

В ближайшем будущем планирую добавить:
1. Полностью рабочую систему формул/скриптов/макросов в рамках основного листа.
2. Полноценная поддержка FoundryVTT ItemPiles листом персонажа и предметов
3. Рабочую систему интеграции расс, заклинаний библиотеками.
4. Версия листа персонажа Монстра, Транспорта, Фракций/Зданий
5. Опциональные окна что можно будет добавлять в лист из подшаблонов, такие как состояние частей тела, более высшие круги магии.

### Для кого этот репозиторий
Если ты тоже играешь по сильно изменённой D&D 5e и тебе нужна максимальная гибкость в настройке листа персонажа этот шаблон может тебе пригодиться.
Можешь свободно форкать, копировать и адаптировать под свои нужды, и буду рад если поделишься своими успехами для улучшения данного шаблона.
